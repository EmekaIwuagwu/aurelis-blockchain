# Aurelis Node

The C++ reference implementation of the Aurelis Blockchain protocol.

## Features
- **Consensus:** Proof of Work (SHA-256d)
- **Architecture:** Modular C++17/20
- **API:** JSON-RPC 2.0
- **Storage:** High-performance block storage

## Build
```bash
mkdir build
cd build
cmake ..
cmake --build .
```
