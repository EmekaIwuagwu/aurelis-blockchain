# Aurelis Wallet

The official desktop and web wallet for the Aurelis Blockchain.

## Features
- **Security:** Local private key management (Encrypted).
- **Design:** Modern UI following Aurelis visual identity (`design/tokens.json`).
- **Connectivity:** Connects to local or remote Aurelis nodes via JSON-RPC.

## Stack
- Frontend: React / Electron (TBD)
- Backend: C++ Core
